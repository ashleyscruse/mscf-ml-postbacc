# ML Postbacc Program: Machine Learning on HPC

A 12-week summer curriculum at the Morehouse Supercomputing Facility (MSCF) that takes post-baccalaureate students from Python fundamentals to deploying machine learning models on production HPC, finishing with an individual capstone project and research poster.

> The program does not stop at ML concepts. Students run training jobs on Vista GPUs, containerize models, and present capstone results as a research poster.

## What This Is

A hands-on summer program for post-baccalaureate students. Everyone starts at a shared Python and data-analysis baseline, moves through the full machine learning workflow, and applies it to an individual capstone on real HPC infrastructure (Vista at TACC).

| Phase | Weeks | What You Walk Away With |
|---|---|---|
| Python Bootcamp | 1 to 3 | Python and data fluency, a package published to PyPI, jobs running on Vista |
| ML Training | 4 to 9 | Models built, evaluated, tuned, and deployed; deep learning on GPUs |
| Capstone | 10 to 12 | An individual project and a research poster, presented at the program close |

## Who This Is For

Post-baccalaureate students preparing for graduate study or industry careers in computational fields. No prior HPC experience is required; the first three weeks bring everyone to a shared Python and data-analysis baseline before the ML curriculum begins.

## The Site

This repository publishes the program site with GitHub Pages (Jekyll, in `docs/`):

- **Schedule** pages: day-by-day plans for Weeks 1 to 3, 4 to 9, and 10 to 12
- **Curriculum** pages: per-topic references for the Python Bootcamp, ML training, and Capstone phases

Live site: https://ashleyscruse.github.io/mscf-ml-postbacc/

## Running Locally

```bash
cd docs
bundle install
bundle exec jekyll serve
```
