## What This Covers

- Installing Python, Anaconda, and Jupyter Notebook
- Variables, types, arithmetic operators, type casting, input and output
- Boolean expressions and logical operators
- Conditionals: `if`, `elif`, `else`
- Loops: `for`, `while`, `range()`, `break`, `continue`
- Data structures: lists, list comprehensions, sets, tuples, dictionaries
- Nested data structures
- Terminal basics: `cd`, `ls`, `pwd`, `mv`, `cp`, navigation
