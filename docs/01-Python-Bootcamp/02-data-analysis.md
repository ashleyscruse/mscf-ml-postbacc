# Data Analysis

*Weeks 1 to 2. Full lesson content is being built out. See the [Weeks 1 to 3 schedule](../Schedule/weeks-1-3) for the day-by-day plan.*

## What This Covers

- NumPy: arrays, indexing, slicing, vectorized operations vs. loops, reshaping
- Pandas: Series and DataFrames, loading CSVs, filtering, sorting
- `.loc` / `.iloc`, `value_counts`, `groupby`, aggregation
- Handling missing data: `.isnull()`, `.fillna()`, `.dropna()`
- Visualization with Matplotlib: line, bar, scatter, histograms
- Visualization with Seaborn: boxplots, heatmaps, pairplots
- Calling APIs with `requests`, parsing JSON, converting responses to DataFrames
- Error handling with `try` / `except` and debugging strategies
