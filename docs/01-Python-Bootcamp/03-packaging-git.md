# Packaging, Git, and PyPI

*Week 2. Full lesson content is being built out. See the [Weeks 1 to 3 schedule](../Schedule/weeks-1-3) for the day-by-day plan.*

## What This Covers

- Defining and calling functions: arguments, return values, scope
- Writing reusable, modular code
- Python package structure: `__init__.py`, modules
- `pyproject.toml` / `setup.py`
- Git basics: version control concepts, `git init`, `add`, `commit`
- Pushing to GitHub: `git remote`, `git push`
- Publishing to PyPI: `twine upload`, then `pip install` your own package
