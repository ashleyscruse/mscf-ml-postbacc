# TACC and HPC

*Week 3. Full lesson content is being built out. See the [Weeks 1 to 3 schedule](../Schedule/weeks-1-3) for the day-by-day plan.*

## What This Covers

- TACC account creation, SSH key setup, MFA configuration
- First login to Vista
- TACC filesystem tour: `$HOME`, `$WORK`, `$SCRATCH`; capacities and best practices
- Editing files remotely with vim / nano; file permissions (`chmod`)
- Running Jupyter Notebook on Vista compute nodes
- HPC job scheduling fundamentals
- Slurm: `sbatch`, `squeue`, `scancel`; writing batch job scripts
- Submitting and monitoring Python and data-processing jobs on Vista
- Checking job status and GPU usage (`nvidia-smi`)
