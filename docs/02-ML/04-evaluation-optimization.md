# Model Evaluation and Optimization

*Week 7. Full lesson content is being built out. See the [Weeks 4 to 9 schedule](../Schedule/weeks-4-9) for the day-by-day plan.*

## What This Covers

- Evaluation deep dive: precision, recall, F1-score, classification reports
- ROC curves and AUC
- Regression metrics: MAE, MSE, RMSE, R squared
- Overfitting vs. underfitting: the bias-variance tradeoff
- Cross-validation: k-fold and stratified k-fold
- Hyperparameter tuning: Grid Search and Random Search (`GridSearchCV`)
- Regularization: L1, L2, ElasticNet
